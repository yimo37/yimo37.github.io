Lively Wallpaper的使用教程

各位看官，想不想和我一样有这样一个漂亮的动态桌面

1.下载软件lively wallpaper的安装包
（这个软件我会以链接的形式放在最后）

2.进行安装

点击下一步后进行安装

注意：
安装进度条满后会让你安装两个软件，Microsoft Visual c++ 2015-2019 Redistributable和Microsoft Windows Desktop Runtime - 3.1.7(x86)。安装程序之前，你可以去设置->应用->应用与功能中中查找自己安装这两个软件了没，如果安装了就可以跳过安装这两个软件。

(这个是我已经装好的如果需要安装的话的话就一步一步往下走就行了）

3.打开应用并添加壁纸
壁纸一般自己可以在网上找的

点击加号

点击预览在选择文件里找到你要上传的文件

点级右下角的打开
在主界面就会显示配置文件，可以对其定义，点击ok即可

4.壁纸声音的处理
一般来说视频都是有声音的所以你的壁纸也是会有声音的
所以我们要对声音进行设置
点击设置—>音频，通过控制主音量就可以了
